Componente React + TypeScript para Cámara y Subida de Fotos
Prompt principal para React (versión completa):
text
Actúa como experto frontend senior especializado en React 18+, TypeScript, hooks modernos y manejo de archivos. Necesito desarrollar un componente React para capturar y subir fotos que se integre con un backend Node.js/Express.

## CONTEXTO TÉCNICO
- **Frontend**: React 18+ con TypeScript
- **Build tool**: Vite (preferido) o Create React App
- **Estado**: React Hooks (useState, useEffect, useRef, useCallback)
- **HTTP Client**: Axios o Fetch API
- **UI Framework**: Tailwind CSS (opcional, puede ser CSS modules o styled-components)
- **Backend**: Node.js + Express con endpoint POST /api/upload
- **Formato de subida**: multipart/form-data

## REQUISITOS DEL COMPONENTE REACT

### 1. COMPONENTE PRINCIPAL: `<CameraUpload />`
**Props (TypeScript interface):**
```typescript
interface CameraUploadProps {
  userId?: string;           // ID opcional para asociar foto a usuario
  onUploadSuccess?: (response: UploadResponse) => void;
  onUploadError?: (error: Error) => void;
  maxSizeMB?: number;        // Tamaño máximo en MB (default: 1)
