# Tech Stack

## Backend (API)

- **Runtime:** Node.js
- **Framework:** Express
- **Language:** TypeScript
- **Database:** MongoDB (with Mongoose ODM)
- **Authentication:** JWT (jsonwebtoken)
- **Testing:** Jest

## Frontend (Client)

- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Routing:** React Router
- **HTTP Client:** Axios
- **Charts:** Chart.js + react-chartjs-2
- **Styling:** SCSS
- **Testing:** Vitest

## Project Structure

```
├── api/          # Node.js + Express API
├── client/       # React + Vite frontend
└── PROJECT.md    # This file
```
